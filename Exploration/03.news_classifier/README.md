### 추가로 복습할 내용

1. 데이터 늘려보기 : 크롤링을 통해서 데이터의 수를 늘린다
2. 다른 Classifier도 사용
   - 이번에 복습을 하면서 classifier들의 특징 및 차이점들을 명확하게 정리하고 넘어간다
3. 데이터를 늘린 후, **데이터의 균형**, **불용어**가 얼마나 영향을 끼치는지 더 확인해본다.
